Research
Embodied AI · Navigation

SeM-H: Semantic Mapless Homing

Enabling mobile robots to navigate unknown environments using frozen Vision-Language Models as implicit world models. The robot reasons about where it is by what it sees — no geometric maps, no runtime learning. Uses Bayesian belief filtering over semantic room identity, with LLM-generated breadcrumb chains for multi-room navigation.

Computer Vision · Robotics

CV Pipeline for Autonomous Refuelling

Occlusion-robust vehicle detection for autonomous refuelling robot arms. Built a synthetic occluded image dataset and benchmarked YOLO, RF-DETR (ViT), and YOLOv11-OCSA. Key finding: attention-based models maintain robustness under occlusion where CNNs degrade. YOLOv11-OCSA achieved best F1 and fastest inference.
