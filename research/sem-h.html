The Core Idea

SeM-H enables a mobile robot to navigate unknown environments without ever building a geometric map. Instead of measuring walls and corners, the robot navigates by understanding what it sees — recognising objects, reasoning about room identity, and following chains of semantic landmarks.

Pretrained Vision-Language Models already contain an implicit understanding of the world. SigLIP knows that fridges belong in kitchens. SeM-H exploits this frozen knowledge as a navigation prior — no runtime learning, no mapping from scratch.

LiDAR-based SLAM builds dense geometric maps. SeM-H uses VLM-driven semantic understanding instead.

Where SeM-H Sits

Existing systems each make one of two compromises: deep reasoning at high compute cost (DyNaVLM), or lightweight control with shallow semantics (text2nav). SeM-H targets the gap — deep semantic priors with a reactive, lightweight control loop.

The Decision Cascade

At each timestep the robot picks the simplest sufficient level of reasoning — the pyramid narrows downward as context becomes clearer.

4

Exploration — "I have no idea where I am"

Room belief is near-uniform (high entropy). Perform 360° semantic scan to establish room identity, then drop to Level 3.

3

Semantic Navigation — "Wrong room, plan a path"

HTP generates a distributional breadcrumb chain. Robot advances chain by maximising observation-expectation overlap at each step.

2

Local Semantic Search — "Right room, can't see it"

Rotate in place, scan with SigLIP, move toward regions of highest cosine similarity to the target text embedding.

1

Direct Visual Homing — "I see it, go to it"

P-SVO produces heatmap centroid → BBS smooths with Kalman filter → PHC steers. Pure reactive servoing. No planning, no room reasoning.

Object-Set Bayesian Localisation

Holistic scene classification is ambiguous — a bedroom with a table might score 30% living room. SeM-H decomposes the scene into individual objects and uses the object set as evidence in a Bayesian room-belief filter. Temporal accumulation of evidence is what makes this sharp.

The Semantic Filter

SeM-H applies recursive Bayesian estimation in a semantic state space rather than a geometric one. The novelty is in what gets filtered — not the mathematics.

1. Inject diffusion 2. Detect objects 3. Bayes update

Dual-Purpose Perception

SigLIP produces two outputs from a single forward pass: a patch-level heatmap for visual servoing ("where is the target?"), and an object detection vector for room localisation ("what am I looking at?"). OWL-ViT serves only as an occasional landmark verifier, not in the main loop.

SigLIP achieves 74% navigation success vs CLIP's 62%, with 26.5% better spatial sensitivity via sigmoid loss.

Hardware

Engineered for deployment on a TurtleBot3 Burger with Raspberry Pi 4 (4GB). SigLIP base (86M params) fits within Pi 4's memory constraints. LiDAR is maintained strictly as a safety obstacle layer — not used for SLAM. All simulation work uses Gazebo with the AWS RoboMaker house environment.

TurtleBot3 Burger + OAK-D Pro (RGB-D + VPU) + LDS-01 LiDAR + Raspberry Pi 4 + OpenCR 1.0.
